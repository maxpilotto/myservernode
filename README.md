# Getting started
```shell
npm install
npm start
```

# Routes
## `/api/v1/info`
Shows infos about the v1 API 

## `/api/v1/todos/:api_key`
Shows all the todos for that API key

## `/api/v1/todos/:api_key/:id`
Shows the todo for that API key and that id
